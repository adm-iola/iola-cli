# Архивы и мастер настройки

`iola-cli` использует 7-Zip как штатный архиватор. Если 7-Zip не найден, `iola archive doctor` и `iola onboard` устанавливают его автоматически.

Поддерживаемые форматы зависят от 7-Zip:

- `.zip`
- `.7z`
- `.rar`
- `.tar`
- `.gz`
- `.tgz`
- `.bz2`
- `.xz`
- другие форматы, которые поддерживает установленный 7-Zip

Команды:

```bash
iola archive doctor
iola archive list docs.zip
iola archive test docs.zip
iola archive extract docs.zip --output ./out
iola archive create docs.zip ./docs
iola archive index docs.zip
```

Индекс архива:

```bash
iola index archive docs.zip
iola index search "контракт"
```

Мастер настройки:

```bash
iola onboard
```

Мастер проверяет Node/npm, SQLite, API и 7-Zip.

В списке компонентов есть отдельный пункт `7-Zip / архивы`. Если выбрать его, CLI установит архиватор сразу из мастера настройки.

Мастер также предлагает подключить workspace, policy, Ollama, OpenAI, OpenRouter, Codex CLI, MCP, browser runtime, Госуслуги и индекс локальных документов.

Подробное описание пунктов мастера вынесено на страницу [Мастер настройки](Мастер-настройки).
