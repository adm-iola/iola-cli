# Контекст и память

Контекстные файлы позволяют закрепить правила конкретного проекта:

```bash
iola context init
iola context list
iola context show
```

Поддерживаются файлы:

- `IOLA.md`;
- `.iola/context.md`.

Память хранится локально в SQLite:

```bash
iola memory show
iola memory add "Пользователь предпочитает краткие ответы"
iola memory suggest
iola memory approve 1
iola memory reject 1
```

CLI может предлагать новые записи памяти, но не добавляет их в рабочую память без подтверждения пользователя.
