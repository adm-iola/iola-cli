# Commands

Основные команды:

```bash
iola --help
iola init
iola doctor
iola doctor --fix
iola update
iola version --check
iola config validate
iola config schema
```

Данные:

```bash
iola search "Петрова"
iola search "Петрова" --local --fts
iola schools --limit 10
iola kindergartens --search "29"
iola card "школа 29"
iola data schools --format csv --output schools.csv
```

Локальная БД:

```bash
iola db status
iola sync
iola sync status
iola diff
iola quality
iola history search "Петрова"
iola sessions search "Петрова"
iola sessions compact 1
```

AI:

```bash
iola ask "найди школу 29"
iola ask "найди школу 29" --schema json
iola ask "найди школу 29" --bare --quiet
iola agents list
iola agents run quality-checker "проверь школы"
iola skills list
iola tools toolsets
iola files status
iola files mode read-only
iola files tree .
iola files search "Петрова" --path .
iola policy use analyst
iola tasks list
iola artifacts list
iola snapshot list
iola trace last
iola export education-contacts --format xlsx --output contacts.xlsx
iola changes list
iola import file data.csv --dataset custom
iola index folder ./docs
iola reports list
iola plugins list
iola mcp serve
iola memory suggest
```

Контекст, расписания и daemon:

```bash
iola context init
iola context list
iola cron add "каждый день 09:00 -- quality"
iola cron tick
iola daemon status
iola daemon start
iola rpc call status
iola rpc call search --query Петрова --dataset schools
```

Интерактивный режим:

```bash
iola agent
```
